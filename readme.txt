Apache с включенным mod_rewrite. PHP не ниже 5.6. MySQL/MariaDB любой версии

Для установки необходимо в папке с проектом запустить composer install

Для миграции данных тестовой базы использовать файл bit.sql
Для настройки подключения к базе данных необходимо отредактировать файл src/config/config.php в секции connection

Корневой папкой сервера указать папку public.

Логин - rootroot
Пароль - rootroot

Для ввода средств "источник пополнения" надо указывать: r1-r99
Для вывода средств "куда вывести" надо указывать w1-w99
